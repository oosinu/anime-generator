# 角色设计模板

## YAML 头部

```yaml
---
style: classic
character_count: 3
---
```

## 主角模板

```markdown
## 主角：[角色名]

### 基本信息
- **年龄**：[年龄]
- **性别**：[性别]
- **身份**：[职业/身份]
- **故事弧线**：[开始状态] → [转变] → [结束状态]

### 视觉设计
- **身高体型**：[描述]
- **发型发色**：[详细描述]
- **面部特征**：[眼睛、鼻子、表情特点]
- **标志性特征**：[独特识别特征]

### 服装设计
- **常服**：[日常服装描述]
- **正装**：[正式场合服装，如有]
- **色彩搭配**：[主色调]

### 表情库
- **日常**：[默认表情]
- **开心**：[开心表情描述]
- **愤怒**：[愤怒表情描述]
- **悲伤**：[悲伤表情描述]
- **思考**：[思考表情描述]

### 关键时刻外观
- **[场景1]**：[该场景特殊外观]
- **[场景2]**：[该场景特殊外观]
```

## 配角模板

```markdown
## 配角：[角色名]

### 基本信息
- **身份**：[与主角的关系/职业]
- **作用**：[在故事中的功能]

### 视觉设计
- **外观摘要**：[简要描述]
- **标志性特征**：[快速识别特征]
- **服装**：[主要服装]
```

## 角色关系图

```
主角
├── 导师/朋友
├── 对手/挑战
└── 支持者
```

## 视觉一致性检查清单

- [ ] 角色比例一致
- [ ] 面部特征可识别
- [ ] 发型全篇一致
- [ ] 服装颜色一致
- [ ] 标志性特征每次出现
