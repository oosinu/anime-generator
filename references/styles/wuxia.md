# Wuxia - 武侠水墨风格

name: wuxia
description: 中国武侠水墨风格，写意留白，江湖气息

## 核心特征
- 水墨画质感
- 飞白留白
- 动态武打场景
- 诗意氛围
- 传统中国美学

## 线条规范
- 毛笔质感的线条
- 粗细变化自然
- 飞白效果
- 墨色浓淡变化

## 角色设计
- 传统服饰（汉服、武侠装）
- 飘逸的衣袖和头发
- 内敛但有力的表情
- 武器设计精细

## 场景设计
- 山水画背景
- 古建筑（亭台楼阁）
- 竹林、瀑布、悬崖
- 月光、雾气、飘雪

## 色彩规范
primary: "#1A202C"  # 墨黑
accent: "#C53030"   # 朱红
secondary: "#718096" # 灰
background: "#F7FAFC" # 宣纸白
gold: "#D69E2E"     # 金

## 动作表现
- 武打动作夸张但优美
- 内力/气场可视化
- 速度线和冲击波
- 定格关键动作

## 适用内容
- 武侠小说改编
- 中国历史
- 仙侠故事
- 传统文化介绍
