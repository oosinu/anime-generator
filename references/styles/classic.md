# Classic - Ligne Claire 传统欧漫风格

name: classic
description: 传统 Ligne Claire 风格，清晰线条，平涂色彩

## 线条规范
- 均匀、干净的轮廓线（约 2px）
- 不使用阴影线或交叉阴影
- 所有元素使用黑色墨水轮廓
- 阴影通过平涂色块表现

## 角色设计
- 轻度风格化/卡通化，但比例写实
- 独特可辨的面部特征
- 表情清晰，情感明确
- 服装符合时代特征
- 角色外观全篇一致

## 背景处理
- 详细写实的背景，建筑精确
- 时代相关的道具和技术
- 清晰的空间深度和透视
- 环境细节讲述故事
- 简化角色与详细背景形成对比

## 色彩规范
primary: "#3182CE"  # 蓝
accent: "#E53E3E"   # 红
secondary: "#ECC94B" # 黄
skin: "#F7CFAE"     # 肤色
background: "#FFFAF0" # 米白

## 色彩方法
- 平涂色彩，无渐变（纯正 Ligne Claire）
- 每页使用有限调色板
- 色彩支持叙事情绪
- 场景内光照逻辑一致

## 适用内容
- 人物传记
- 历史故事
- 教育内容
- 平衡叙事
