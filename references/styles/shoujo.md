# Shoujo - 少女漫画风格

name: shoujo
description: 少女漫画风格，浪漫梦幻，适合情感主题

## 核心特征
- 大眼睛闪亮
- 柔和线条
- 花朵装饰
- 浪漫氛围
- 梦幻效果

## 线条规范
- 纤细优美的线条
- 睫毛细节丰富
- 头发飘逸
- 服装褶皱精细

## 装饰元素
- 花瓣飘落
- 星光闪烁
- 泡泡效果
- 心形元素
- 闪光特效

## 色彩规范
primary: "#ED64A6"  # 粉红
accent: "#9F7AEA"   # 紫
secondary: "#F687B3" # 浅粉
background: "#FFF5F7" # 粉白
highlight: "#FFFFFF" # 高光

## 适用内容
- 情感故事
- 校园生活
- 友情主题
- 成长故事
- 浪漫内容
