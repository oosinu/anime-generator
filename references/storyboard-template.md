# 分镜脚本模板

## YAML 头部

```yaml
---
title: "标题"
style: classic
layout: standard
aspect_ratio: "3:4"
page_count: 12
narrative_approach: chronological  # chronological / thematic / character
---
```

## 封面设计

```markdown
## 封面

**视觉描述**:
- 主视觉：主角站立于标志性场景
- 标题位置：上方居中
- 风格要素：[风格特有元素]

**文字元素**:
- 标题：《书名》
- 副标题：[如有]
- 作者/来源：[如有]
```

## 内容页模板

```markdown
## 第 N 页：[页面主题]

### 分镜 1（左上，大）
**场景描述**: [详细场景]
**角色动作**: [角色行为]
**对话/旁白**:
- 角色A："对话内容"
- 旁白：[旁白内容]
**情绪**: [场景情绪]

### 分镜 2（右上，小）
**场景描述**: [详细场景]
**角色动作**: [角色行为]
**对话/旁白**:
- 角色B："对话内容"
**情绪**: [场景情绪]

### 分镜 3（下方，宽）
...
```

## 视觉提示规范

每个分镜应包含：
1. **场景描述**：环境、时间、氛围
2. **角色动作**：姿势、表情、互动
3. **对话/旁白**：气泡内容或旁白
4. **情绪**：该分镜想传达的情绪
5. **构图建议**：远景/中景/近景/特写

## 叙事节奏

| 页面位置 | 建议内容 |
|---------|---------|
| 第1页 | 开场、背景介绍 |
| 第2-3页 | 人物登场、问题浮现 |
| 中间页 | 主要情节发展 |
| 倒数第2页 | 高潮或转折 |
| 最后一页 | 结局、总结、留白 |
